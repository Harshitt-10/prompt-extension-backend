# Privacy Policy – Prompt Enhancer Extension

This Chrome extension helps users generate improved prompts for ChatGPT.

## Data Collection
The extension collects the prompt text entered by the user in the extension interface in order to generate an improved prompt using an AI service.

## How Data Is Used
The entered prompt text is sent to a backend server which processes the request using an AI model and returns a generated prompt.

## Data Storage
No user data is stored, logged, or shared.

## Third-Party Services
Prompt text may be processed by an AI inference service through the backend server.

## Personal Information
This extension does not collect personally identifiable information such as name, email, address, or payment details.

## Changes
This privacy policy may be updated if the extension functionality changes.

## Contact
For questions regarding this privacy policy, please contact the developer.
